## String

## Sequences(序列)
```
list 列表:动态存储,可修改,有序有索引,可增删改查
tuple 元组:固定存储,不可修改,有序有索引
range 范围:固定存储,不可修改,有序有索引
* enumerate(枚举,带序号) → 便利的时候枚举
```

## Sets集合
```
set 无序、不重复、可变(常用于存储,去重)
frozenset 无序、不重复、不可变(常作为字典)
```

## Mappings映射(类似JS中的对象)

```
dict 字典/对象类型
# 使用types库,可以更安全,提供只读的数据结构
```

| 分类                | Python                                                       | JavaScript                                          | 说明                                                         |
| ------------------- | ------------------------------------------------------------ | --------------------------------------------------- | ------------------------------------------------------------ |
| **Number**          | `int`, `float`, `complex`                                    | `number`（整型/浮点型同一类型）, `BigInt`           | Python 区分整型/浮点/复数，JS 只有 number（双精度浮点）+ BigInt |
| **Boolean**         | `bool` (`True/False`)                                        | `boolean` (`true/false`)                            | 一致                                                         |
| **String**          | `str`                                                        | `string`                                            | 一致，Python `str` 是 Unicode                                |
| **Sequence (序列)** | `list`（动态数组，可修改，有序）<br />`tuple`（不可变，有序）<br />`range`（不可变范围）<br />`enumerate`（枚举器） | `Array`（动态数组，可修改，有序）                   | Python 区分 list/tuple/range，JS 只有 Array                  |
| **Set (集合)**      | `set`（可变，无序，不重复）<br />`frozenset`（不可变，无序，不重复） | `Set`（可变，不重复）、无内置不可变集合             | Python 有可变/不可变两种，JS 只有 Set                        |
| **Mapping (映射)**  | `dict`（字典，键值对），可用 `MappingProxyType` 只读         | `Object`（键值对），`Map`（键值对，支持任意类型键） | Python 的 dict ≈ JS 的 Object/Map                            |
| **Null / 空值**     | `None`                                                       | `null`, `undefined`                                 | Python 只有 `None`，JS 区分 `null` 和 `undefined`            |
| **Function**        | `def` 定义函数<br />`lambda` 匿名函数                        | `function` 声明<br />`()=>{}` 箭头函数              | 类似，Python `lambda` 只能写简单表达式                       |
