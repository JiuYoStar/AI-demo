# 🐍 Python f-string 常用语法速查表

## 1. 基础用法

```python
x = 42
name = "GG"

print(f"值: {x}")
print(f"表达式: {2+3}")
print(f"函数调用: {len('abc')}")
```

---

## 2. 转换符

```python
text = "hi\n你好"
print(f"!s => {text!s}")  # str()
print(f"!r => {text!r}")  # repr()
print(f"!a => {text!a}")  # ascii()
```

---

## 3. 对齐与宽度

```python
print(f"左对齐: {'hi':<5}")
print(f"右对齐: {'hi':>5}")
print(f"居中: {'hi':^5}")
print(f"填充: {7:0>4}")     # 0007
```

---

## 4. 数字格式化

```python
print(f"小数位: {3.14159:.2f}")   # 3.14
print(f"百分比: {0.123:.2%}")     # 12.30%
print(f"千位分隔: {1234567:,}")   # 1,234,567
```

---

## 5. 进制 & 科学计数法

```python
print(f"二进制: {255:b}")   # 11111111
print(f"八进制: {255:o}")   # 377
print(f"十六进制: {255:x}") # ff
print(f"科学计数: {12345:e}") # 1.234500e+04
```

---

## 6. 动态格式

```python
width, prec = 8, 3
print(f"{3.14159:{width}.{prec}f}")  # 宽度和精度由变量控制
```

---

## 7. 多行 f-string

```python
name = "GG"
msg = f"""
Hello {name},
This is a multi-line f-string.
"""
print(msg)
```

# 🐍 Python f-string 对齐 & 补位速查表

## 📌 基本语法

```python
f"{value:{fill}{align}{width}}"
```

* **fill**：填充字符（任意一个字符）
* **align**：对齐方式

  * `<` 左对齐
  * `>` 右对齐（默认）
  * `^` 居中对齐
  * `=` 符号靠左，其余补齐（数字专用）
* **width**：总宽度

---

## 🔹 示例

```python
print(f"[{42:>5}]")    # 右对齐，空格补齐 → [   42]
print(f"[{42:0>5}]")   # 右对齐，0 补齐 → [00042]
print(f"[{42:.>5}]")   # 右对齐，. 补齐 → [...42]
print(f"[{42:*^7}]")   # 居中对齐，* 补齐 → [**42***]
print(f"[{-42:0=5}]")  # 符号靠左，0 补齐 → [-0042]
```

---

## 📌 对齐 & 补位对比表

| 格式    | 效果        | 示例               | 输出          |
| ----- | --------- | ---------------- | ----------- |
| `>5`  | 右对齐，补空格   | `f"[{42:>5}]"`   | `[   42]`   |
| `<5`  | 左对齐，补空格   | `f"[{42:<5}]"`   | `[42   ]`   |
| `^5`  | 居中对齐，补空格  | `f"[{42:^5}]"`   | `[ 42  ]`   |
| `0>5` | 右对齐，补零    | `f"[{42:0>5}]"`  | `[00042]`   |
| `.>5` | 右对齐，补点    | `f"[{42:.>5}]"`  | `[...42]`   |
| `*^7` | 居中对齐，补星号  | `f"[{42:*^7}]"`  | `[**42***]` |
| `0=5` | 符号靠左，数字补零 | `f"[{-42:0=5}]"` | `[-0042]`   |

---

## 📌 总结

* **默认补空格**（右对齐）
* **fill 可自定义**：可以是 `0`、`.`、`*`、`-` 等任意字符
* **align 控制方向**：`<` 左，`>` 右，`^` 居中，`=` 符号靠左（数字）
