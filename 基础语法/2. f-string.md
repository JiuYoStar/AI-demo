# f-string 速查表



| 功能 | 语法示例 | 输出结果 | 说明 |
| :--- | :--- | :--- | :--- |
| **基础** | `f"{x}"` | `42` | 变量插值 |
| **调试 (3.8+)** | `f"{x=}"` | `x=42` | 自带变量名打印 |
| **简易计算** | `f"{1+1}"` | `2` | 表达式求值 |
| **转换符** | `f"{'hi'!r}"` | `'hi'` | `!r` 调 repr(完整性), `!s` 调 str(可读性) |
| **左对齐** | `f"{'hi':<5}"` | `'hi   '` | 宽度为 5 |
| **右对齐** | `f"{'hi':>5}"` | `'   hi'` | 默认左补空格 |
| **居中对齐** | `f"{'hi':^5}"` | `' hi  '` | 两侧补空格 |
| **填充对齐** | `f"{7:0>4}"` | `0007` | 字符填充，如 `0` |
| **符号固定** | `f"{-7:0=4}"` | `-007` | 符号左对齐，数字右对齐 |
| **小数位数** | `f"{3.1415:.2f}"` | `3.14` | 保留 2 位小数 |
| **百分比** | `f"{0.123:.1%}"` | `12.3%` | 自动乘 100 并加 % |
| **千分位** | `f"{1234567:,}"` | `1,234,567` | 金额常用分隔符 |
| **进制转换** | `f"{255:x}"` | `ff` | `b`二进制, `o`八进制, `x`十六进制 |
| **动态格式** | `f"{3.14:{w}.{p}f}"` | - | 嵌套变量控制宽度/精度 |

---

### Tips
*   **大括号转义**：`f"{{ }}`" 输出 `{ }`。
*   **多行支持**：`f"""多行文本 {x}"""`。
*   **优先对齐**：格式说明符跟随在 `:` 之后。
