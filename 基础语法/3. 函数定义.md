# 函数定义速查

## 1. 核心对比

| 功能 | Python | JS 对应 / 差异 |
| :--- | :--- | :--- |
| **标准定义** | `def name(arg):` | `function name(arg) {}` |
| **匿名函数** | `lambda x: x + 1` | `(x) => x + 1` (仅限单行表达式) |
| **默认参数** | `def f(a=1):` | `function f(a=1) {}` (基本一致) |
| **不定长位置参数** | `*args` | `...args` (Rest 参数) |
| **不定长关键字参数** | `**kwargs` | JS 无直接对应 (类似解构对象) |
| **仅限关键字参数** | `def f(*, a):` | JS 无直接对应 |

## 2. 关键代码示例

### 基础定义
```python
def greet(name, msg="Hello"):
    """这是 docstring (类似 JS 中的注释文档)"""
    return f"{msg}, {name}!"
```

### 匿名函数 (lambda)
*   **局限性**：只能写一行表达式，不能包含 `return`、多行逻辑。
```python
square = lambda x: x * x
# 对应 JS: const square = x => x * x;
```

### args 与 kwargs
```python
def debug_func(*args, **kwargs):
    # args 是元组 (tuple): (1, 2)
    # kwargs 是字典 (dict): {'key': 'val'}
    pass

debug_func(1, 2, key="val")
```

## 3. 必知差异
1.  **缩进**：Python 强依赖缩进代替 `{}`。
2.  **没有变量提升**：函数必须先定义后调用（处于同一作用域时）。
3.  **返回空值**：如果没有 `return`，默认返回 `None` (JS 返回 `undefined`)。



| 顺序 | 参数类型       | 示例     | 作用                                          |
| ---- | -------------- | -------- | --------------------------------------------- |
| 1    | 位置参数       | a, b     | 最基本的必填参数                              |
| 2    | 可变位置参数   | *args    | 接收多出来的所有位置参数，打包成元组 📦 tuple  |
| 3    | 关键字参数     | c=1      | 有默认值的参数                                |
| 4    | 可变关键字参数 | **kwargs | 接收多出来的所有关键字参数，打包成字典 📖 dict |
