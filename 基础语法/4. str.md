# String (str) 常用 API 速查

| 分类 | 常用方法 | 说明 |
| :--- | :--- | :--- |
| **切片 (Slicing)** | `s[0]`, `s[-1]`, `s[1:4]`, `s[::-1]` | 索引、范围切片、**反转** |
| **查 (Query)** | `s.find(t)`, `s.index(t)`, `s.count(t)` | find 找不到返 -1，***index 报错*** |
| **大小写** | `s.upper()`, `s.lower()`, `s.title()`, `s.capitalize()` | 全部大写/小写/标题/首字母大写 |
| **修剪 (Trim)** | `s.strip()`, `s.lstrip()`, `s.rstrip()` | 去除两端/左/右空白或指定字符 |
| **替换 (Replace)** | `s.replace(old, new, [n])` | 替换，可限制前 n 个 |
| **拆分 (Split)** | `s.split([sep])`, `s.splitlines()` | 默认按空格切；JS 的 `s.split("")` 在 PY 中用 `list(s)` |
| **合并 (Join)** | `sep.join(list)` | 将序列合并为字符串 (如 `",".join(['a','b'])`) |
| **判断 (Check)** | `s.startswith()`, `s.endswith()`, `s.isdigit()`, `s.isalpha()` | 返回布尔值 |
| **对齐** | `s.center(w, fill)`, `s.zfill(w)` | 居中对齐、左侧补 0 |

---

### JS 开发者注意
*   **不可变性**：所有方法均返回**新字符串**，原字符串不变。
*   **单字符切分**：`s.split("")` 在 Python 中无效，请使用 `list(s)` 或 `[c for c in s]`。
*   **成员判断**：推荐使用 `'val' in s` 替代 `find() != -1`。
