# 🐍 Python `str` 常用 API 速查表



## 1. 创建 & 拼接

```python
s = "hello"
t = "world"

# 拼接
s + t          # "helloworld"
" ".join([s,t]) # "hello world"
```

---

## 2. 查 (查找 & 访问)

```python
s = "hello world"

s[0]         # 'h'  (索引)
s[-1]        # 'd'  (倒数)

s.find("lo") # 3   (找不到返回 -1)
s.index("lo")# 3   (找不到会报错)

s.rfind("o") # 7   (从右查找)
s.count("o") # 2
```

---

## 3. 改 (替换 & 转换)

```python
s.replace("world", "Python") # "hello Python"
s.upper()    # "HELLO WORLD"
s.lower()    # "hello world"
s.title()    # "Hello World"
s.capitalize() # "Hello world" # 首字母大写
s.swapcase() # "HELLO WORLD" -> "hello world"
```

---

## 4. 删 (去除)

```python
s.strip()    # 去掉首尾空白
s.strip("hd")# 去掉首尾指定字符
s.lstrip()   # 去掉左边
s.rstrip()   # 去掉右边
# replace也可以对某些字符串进行删除操作, 等于js中的replaseAll
# replace(old, new, number) 可以限制修改的次数
```

---

## 5. 判断 (返回 bool)

```python
s.startswith("he")  # True
s.endswith("ld")    # True

s.isalpha()   # 是否全字母
s.isdigit()   # 是否全数字
s.isalnum()   # 是否字母+数字
s.isspace()   # 是否空白
s.islower()   # 是否小写
s.isupper()   # 是否大写
```

---

## 6. 分割 & 拼接

```python
s.split()      # ['hello', 'world'] (默认空格)
s.split("o")   # ['hell', ' w', 'rld']
".".join(["a","b","c"]) # "a.b.c"

# py的split方法,不会将所有的字符分割, 需要使用list
'hello'.split() # → ['hello']
list('hello') # →  ['h', 'e', 'l', 'l', 'o']
print([ch for ch in 'hello']) # →  ['h', 'e', 'l', 'l', 'o']

```

---

## 7. 对齐 & 填充

```python
s.center(20, "-") # "-----hello world-----"
s.ljust(20, ".")  # "hello world........."
s.rjust(20, ".")  # ".........hello world"
s.zfill(20)       # "0000000000hello world"  支持一个参数, 从左侧填充
```

---

## 8. 反转

```python
# str[start:stop:step] +正序 -倒序  step间隔几步取值
s[::-1] # 'dlrow olleh'
s[::-2] # 'drwolh'
s[4:1:-1] # 'oll'

s[::-1] === s[None:None:-1]
start: None -> 默认结尾
stop: None -> 默认开头
所以有反转的效果

```

---

## 9. 插入（无原生，需要拼接）

```python
s = "hello world"
s[:5] + " Python " + s[5:]  # "hello Python  world"
```

---

👉 总结：

* **增/拼接** → `+`, `join`
* **删** → `strip`
* **改** → `replace`, 大小写转换
* **查** → `find`, `index`, `count`, 索引切片
* **判断** →
