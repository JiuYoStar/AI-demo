## 🧮 一、数值模型（Numeric Model）

主要用于各种数学、统计、分页、计算类场景。

| 功能     | 示例                            | 说明     | 典型使用场景                                |
| -------- | ------------------------------- | -------- | ------------------------------------------- |
| 整除     | `9 // 2` → `4`                  | 向下取整 | 分页计算：`page_count = total // page_size` |
| 取余     | `9 % 2` → `1`                   | 余数     | 判断偶数奇数                                |
| 幂       | `pow(2, 3)` → `8`               | 幂运算   | 指数增长                                    |
| 开平方   | `math.sqrt(9)` → `3.0`          | 平方根   | 几何计算                                    |
| 开立方   | `math.cbrt(8)` → `2.0`          | 立方根   | 体积相关计算                                |
| 向上取整 | `math.ceil(9/2)` → `5`          | 向上取整 | 分页页数计算                                |
| 向下取整 | `math.floor(9/2)` → `4`         | 向下取整 | 坐标定位                                    |
| 四舍五入 | `round(3.75, 1)` → `3.8`        | 保留小数 | 金融或UI显示                                |
| 绝对值   | `abs(-10)` → `10`               | 取正值   | 坐标偏移                                    |
| 最大最小 | `max(1, 5, 3)` → `5` / `min(…)` | 极值     | 排名或边界计算                              |
| 平均数   | `sum(nums) / len(nums)`         | 平均     | 数据分析                                    |

## 📜 二、序列模型（Sequence Model）

包括 `list`、`tuple`、`str`，重点在**索引、切片、拼接、反转**。

| 功能     | 示例                    | 说明         | 场景               |
| -------- | ----------------------- | ------------ | ------------------ |
| 索引     | `nums[0]`               | 取第一个元素 | 列表访问           |
| 切片     | `nums[1:4]`             | 取部分序列   | 数据分页、窗口滑动 |
| 拼接     | `a + b`                 | 合并         | 文本拼接、路径合成 |
| 乘法     | `'ab' * 3` → `'ababab'` | 重复序列     | 占位符或分隔符生成 |
| 反转     | `nums[::-1]`            | 翻转         | 排序、镜像         |
| 成员判断 | `'a' in s`              | 判断存在     | 过滤条件           |
| 长度     | `len(s)`                | 元素个数     | 遍历控制           |

## 🧩 三、映射模型（Mapping Model）

即 `dict`，用于键值对操作。

| 功能     | 示例                   | 说明             | 场景         |
| -------- | ---------------------- | ---------------- | ------------ |
| 取值     | `d['key']`             | 通过key取值      | 查配置、缓存 |
| 新增     | `d['x'] = 1`           | 新增键值         | 数据聚合     |
| 删除     | `d.pop('x')`           | 删除             | 清理缓存     |
| 更新     | `d.update({...})`      | 合并             | 动态配置     |
| 遍历     | `for k,v in d.items()` | 遍历键值对       | 数据映射     |
| 安全取值 | `d.get('key', 0)`      | 不存在返回默认值 | 安全访问     |

## 🧮 四、集合模型（Set Model）

无序、唯一的集合类型，适用于集合运算。

| 功能     | 示例     | 说明         | 场景             |      |
| -------- | -------- | ------------ | ---------------- | ---- |
| 并集     | `a | b`  | 合并所有     | 合并标签         |      |
| 交集     | `a & b`  | 共同元素     | 筛选重叠数据     |      |
| 差集     | `a - b`  | 从a中去掉b   | 去重             |      |
| 对称差   | `a ^ b`  | 只存在于一方 | 比较两个结果差异 |      |
| 判断子集 | `a <= b` | a是否在b中   | 权限校验         |      |

## 🧱 五、对象与数据模型（Data Model）

Python 的核心是“**一切皆对象**”，它允许自定义类模拟这些内置行为。

| 功能       | 方法                 | 示例           | 场景         |
| ---------- | -------------------- | -------------- | ------------ |
| 字符串表示 | `__str__`            | `print(obj)`   | 日志输出     |
| 长度       | `__len__`            | `len(obj)`     | 自定义容器   |
| 索引访问   | `__getitem__`        | `obj[i]`       | 模拟数组     |
| 相等比较   | `__eq__`             | `obj1 == obj2` | 数据对比     |
| 加法运算   | `__add__`            | `obj1 + obj2`  | 矩阵、向量   |
| 可迭代     | `__iter__`           | `for x in obj` | 自定义迭代器 |
| 上下文管理 | `__enter__/__exit__` | `with obj:`    | 资源管理     |
| 调用       | `__call__`           | `obj()`        | 模拟函数     |

## 📘 六、分页计算示例（组合运用）

```python
import math

def get_page_info(total_items, page_size, current_page):
    # 页数（向上取整）
    total_pages = math.ceil(total_items / page_size)
    # 数据起止索引
    start = (current_page - 1) * page_size
    end = min(start + page_size, total_items)
    return {
        "total_pages": total_pages,
        "range": (start, end),
        "has_prev": current_page > 1,
        "has_next": current_page < total_pages
    }

# 示例
info = get_page_info(total_items=53, page_size=10, current_page=3)
print(info)
```

输出：

```python
{'total_pages': 6, 'range': (20, 30), 'has_prev': True, 'has_next': True}
```
